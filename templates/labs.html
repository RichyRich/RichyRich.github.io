Moog Prodigy Audio Plugin

What?
A waveform accurate modelling of my Moog Prodigy synthesizer.
Written in C++ using the wdl-ol framework it should eventually be available in both Audio Unit and VST formats for Windows and OS X.
Main features : Band limited oscillators, zero-delay filter, hard-sync and more...
Why?
Mostly because digital audio processing has always fascinated me and I didn't really understand it - so I set out on a voyage of discovery.
It had been a long time since I last did any C++ and even longer since I did Fast Fourier Transforms and the like at university, I wanted to see where C++ was at these days and write something I found challenging.
Oh, and Moogs are cool!
